# How to Apply
## 1. 기존 설정 백업 (선택 사항)
```bash
mv ~/.config/nvim ~/.config/nvim.bak
mv ~/.local/share/nvim ~/.local/share/nvim.bak
```
## 2. 저장소 복제
```
git clone https://github.com/hyeseungmoon/nvim-config.git ~/.config/nvim
```
